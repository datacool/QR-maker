## 📦 가장 쉬운 배포 방법: Netlify Drop

### 단계별 안내:

1. **브라우저에서 다음 주소 열기:**
   ```
   https://app.netlify.com/drop
   ```

2. **`dist` 폴더를 드래그 앤 드롭**
   - 파일 탐색기에서 `dist` 폴더를 찾아서
   - Netlify Drop 페이지로 드래그 앤 드롭

3. **완료!**
   - 몇 초 안에 배포 URL이 생성됩니다
   - 예: `https://amazing-name-123.netlify.app`

### 💡 팁
- 회원가입이나 로그인 **불필요**
- 즉시 배포 가능
- 무료 HTTPS 제공
