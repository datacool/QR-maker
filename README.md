# QR 코드 생성기

React + TypeScript + Tailwind CSS로 만든 로고가 포함된 QR 코드 생성 프로그램입니다.

## 기능

- 여러 링크 입력 (최대 10개)
- 각 링크에 로고 이미지가 중앙에 포함된 QR 코드 생성
- 로고 주변 흰색 배경 처리
- 링크 복사 기능
- QR 코드 일괄 다운로드

## 설치 및 실행

```bash
# 패키지 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 사용 방법

1. 링크를 입력합니다
2. [+추가] 버튼으로 최대 10개까지 링크를 추가할 수 있습니다
3. [생성] 버튼을 누르면 모든 QR 코드가 다운로드됩니다
4. 각 링크 옆의 [복사] 버튼으로 링크를 클립보드에 복사할 수 있습니다
